## How to Play

1. **Select Colors**: Click on colors from the palette to add them to the mixing slots
2. **Mix Colors**: Once you have two colors in the slots, click "Mix Colors!" to create a new pigment
3. **Feed the Crab**: Click "Feed Crab!" to give the mixed pigment to the crab
4. **Watch the Magic**: The crab will change color based on the pigment it eats!

## Running the Game

Simply open `index.html` in your web browser to play the game.

### Using a Local Server (Optional)

For development, you can use any local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

Then open `http://localhost:8000` in your browser.
